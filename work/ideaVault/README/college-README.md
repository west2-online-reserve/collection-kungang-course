# 页面功能与完成内容说明

本页面实现了 福州大学计算机与大数据学院简介页，包括以下内容：

## 页面主要结构

- 顶部导航栏（首页、教师风采、实验室、返回总首页）

- 学院简介介绍段落

- 学院统计数据（教师数、学生数、科研项目等）

- 三张学院相关图片展示与可点击放大预览

- 教师风采展示区（多名教授信息卡片）

- 重点实验室预览区（展示实验室名称、研究方向、位置及负责人）

- 页底返回按钮

## 已实现的交互功能
 
- 图片点击放大预览 Modal

- 教师详情弹窗结构（根据 JS 可拓展）

- 导航栏锚点跳转

- 响应式布局兼容 PC 与移动端

## 使用到的资源与文件

- common.css：页面公共样式

- college.css：学院简介页面专用样式

- common.js：通用交互逻辑

- college.js：图片预览、教师弹窗等逻辑

- Font Awesome 图标库

- 本地图片资源（教学楼、实验室、科研团队）

## 页面亮点

- 排版清晰，内容结构完整

- 图文并茂，支持大图查看

- 教师与实验室信息格式规范，可扩展

- 页面导航顺畅，用户体验良好

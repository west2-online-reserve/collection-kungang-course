# 重点实验室页面功能说明（lab.html）

此页面展示了学院共 27 个重点实验室 的分类、详情及交互功能，帮助用户快速获取实验室的研究方向、地点、负责人和科研成果等信息。

## 页面主要结构

- 顶部导航栏：提供“返回学院简介”按钮

- 页面头部说明：展示实验室总览与简介

- 左侧实验室分类筛选栏

- 全部实验室

  1.  人工智能类（9个）

  2. 网络计算类（6个）

  3. 数据科学类（5个）

  4. 信息安全类（4个）

  5. 集成电路类（3个）

- 右侧实验室列表区域

- 默认显示占位提示

- 点击左侧分类后动态生成卡片列表

- 实验室详情弹窗（Modal）
包含：

  1. 实验室名称

  2. 分类标签

  3. 地点、联系人、联系方式

  4. 研究方向介绍

  5. 科研成果与更多链接

## 已实现的交互功能

- 分类筛选实验室（使用 data-category 自动过滤）

- 点击实验室卡片可打开 详情弹窗

    弹窗包含：

  1. 动态内容填充

  2. 外部链接跳转

  3. 支持 PC 与移动端的 响应式布局

## 使用的前端文件

样式文件

- normalize.css：统一浏览器默认样式

- common.css：全站通用样式

- college.css：学院页面基础样式

- lab-detail.css：（本页专属）优化后的实验室样式

- 脚本文件

- common.js：通用组件逻辑

- lab-data.js：实验室数据源（名称、类别、地点、研究方向等）

- lab-script.js：分类筛选、详情弹窗等动态交互逻辑

## 页面亮点

- 分类清晰，结构专业

- 实验室内容通过 JS 动态加载，更易维护

- 详情弹窗信息全面，体验流畅

- 页面风格统一，符合学院官网设计

# 福大校园打卡点项目说明

## 📌 项目概述

**福大校园打卡点 - 红色点位与特色风景**是一个基于Web的交互式校园打卡系统，专门为福州大学设计，旨在展示校园内的红色教育点位和特色风景地点。该系统结合了地图导航、点位展示、统计分析和打卡记录等功能，为用户提供沉浸式的校园探索体验。

## 🎯 主要功能

### 1. **交互式校园地图**
- 可点击的地图点位标记（红色/蓝色）
- 悬浮显示点位名称
- 3D悬停效果和脉冲动画
- 点击点位查看详细信息

### 2. **打卡点展示与筛选**
- 卡片式打卡点展示
- 四种筛选模式：全部、红色点位、常规点位、热门点位
- 3D悬停卡片效果
- 响应式网格布局

### 3. **统计概览**
- 总打卡点数统计
- 已打卡点数统计
- 打卡完成率计算
- 最受欢迎点位展示
- 动态数字滚动效果

### 4. **打卡功能**
- 本地存储打卡记录
- 实时更新打卡状态
- 打卡成功提示
- 进度追踪

### 5. **详情弹窗**
- 丰富的点位信息展示
- 统计数据显示
- 开放时间、位置等详细信息
- 打卡小贴士

## ✨ 特色功能

### 🎨 **视觉效果**
- **玻璃拟态设计**：卡片、按钮采用毛玻璃效果
- **3D变换效果**：悬停时的3D旋转和位移
- **动态渐变背景**：流动的背景装饰元素
- **粒子动画**：页面背景漂浮粒子效果
- **脉冲标记**：地图点位的呼吸灯效果

### 🔄 **交互体验**
- **水波纹效果**：按钮点击时的涟漪反馈
- **平滑过渡动画**：所有交互都有平滑动画
- **滚动动画**：元素进入视口时的渐显效果
- **图片懒加载**：优化页面加载性能
- **响应式设计**：适配各种屏幕尺寸

### 📊 **数据管理**
- **本地存储**：使用localStorage保存打卡记录
- **实时更新**：所有统计数据实时计算
- **持久化状态**：刷新页面不丢失打卡记录

## 🏗️ 页面结构

### 1. **顶部统计区域**
- 四个统计卡片：总点数、已打卡、完成率、最受欢迎
- 玻璃拟态设计 + 3D悬停效果

### 2. **筛选控制栏**
- 四个筛选按钮：全部、红色、常规、热门
- 点击切换显示对应类型点位

### 3. **校园地图区域**
- 校园地图背景
- 6个可点击点位标记（2红4蓝）
- 悬浮显示点位名称

### 4. **打卡点列表**
- 6个打卡点卡片
- 每张卡片包含：图片、标题、描述、统计数据
- 已打卡状态标记

### 5. **详情弹窗**
- 点击卡片或地图点位打开
- 显示详细信息、统计、打卡按钮

## 🛠️ 技术实现

### **核心技术**
- **HTML5**：语义化结构
- **CSS3**：Flexbox、Grid布局、CSS变量、动画
- **原生JavaScript**：DOM操作、事件处理、本地存储

### **关键特性**
1. **CSS自定义属性**：统一管理颜色和动画参数
2. **CSS Grid/Flexbox**：响应式布局
3. **Intersection Observer**：滚动动画和图片懒加载
4. **CSS 3D变换**：卡片和按钮的3D效果
5. **CSS动画**：关键帧动画实现各种动态效果

### **设计特色**
- **颜色系统**：福州大学红色为主色调，蓝色为辅
- **动效设计**：所有交互都有适当的动画反馈
- **视觉层次**：通过阴影、渐变和透明度创建深度感
- **微交互**：悬停、点击等细节交互丰富

## 📱 使用指南

### 基本操作
1. **查看点位**：点击地图标记或卡片查看详情
2. **筛选点位**：使用顶部筛选按钮过滤点位类型
3. **打卡记录**：在详情弹窗中点击"标记已打卡"
4. **查看进度**：顶部统计卡片显示打卡进度

### 交互提示
- 悬停卡片查看3D效果
- 点击按钮有水波纹反馈
- 地图标记有脉冲动画
- 滚动时元素有渐显动画

# 福州大学红色校史展示网站 README

# 一、项目概述

本项目是福州大学红色校史主题单页展示网站，通过时间轴、红色打卡点、校园地图交互等模块，系统呈现福大1958年建校以来的红色发展脉络与精神传承。网站采用原生前端技术构建，支持响应式布局，适配桌面端、平板及移动端设备。

# 二、功能特点

## 2.1 核心展示功能

- **红色校史时间轴**：按1958年建校奠基、1978年改革传承、2008年校史馆落成、2021年数字化建设、未来展望五大节点分层展示

- **红色打卡点展示**：卡片形式呈现校史馆等打卡点的地址、开放时间及特色

- **校园地图交互**：标注打卡点，支持悬停显简介、点击看详情

- **多媒体预览**：校史及打卡点图片支持点击放大，搭配图文提升效果

## 2.2 交互体验优化

- 详情默认折叠，支持展开/收起，长文本可弹窗放大阅读

- 卡片悬浮阴影、元素渐入动画、导航栏滚动样式变化等动态效果

- ESC键一键关闭所有弹窗及展开面板，锚点跳转支持平滑滚动

## 2.3 视觉设计特色

- 主色调采用福大红色#c8102e与校徽蓝色#0055a4，契合主题

- 定制SVG淡色背景纹理，增强页面层次感

- 响应式适配，移动端时间轴改为单列、导航栏堆叠排列

# 三、技术栈

HTML5（语义化结构）、CSS3（变量管理、Flex/Grid布局、动画）、原生JavaScript（交互实现）、Font Awesome（图标）、SVG（背景纹理）

# 四、目录结构

```plaintext

├── assets/
│   ├── css/（通用样式、时间轴样式）
│   ├── images/（校史、打卡点、地图图片）
│   └── js/（外部JS存放目录，可选）
├── index.html（网站主页面）
└── README.md（项目说明文档）
```

# 五、页面结构说明

包含固定顶部导航栏（首页、红色校史等4个导航项）、居中标题区域、左右交替/单列布局的时间轴核心区、红色背景的打卡点展示区、统一管理的弹窗组件及版权信息页脚。

# 六、部署与使用

## 6.1 部署与访问

将项目文件上传至Web服务器根目录，确保路径正确；直接访问服务器域名/IP，或本地打开index.html预览。

## 6.2 交互操作

图片点击放大、详情按钮控制内容展收、长文本点击弹窗阅读、地图点位悬停/点击交互、ESC键关闭弹窗及展开内容。

# 七、自定义指南

- 内容更新：复制现有节点/卡片结构，修改年份、图片、文本等信息

- 样式调整：修改CSS根变量--fd-red/--fd-blue可快速替换主题色

- 功能扩展：可结合后端接口实现动态加载，或添加留言、分享功能

# 八、浏览器兼容

兼容Chrome、Firefox、Safari、Edge最新版本；支持IE11及以上（需调整部分CSS3属性）。

# 九、开发困难与解决方案

|核心问题|解决方案|
|---|---|
|响应式时间轴布局错乱|设768px/1200px双断点，用CSS变量控制间距，伪元素定位保证时间轴线连续|
|地图点位坐标偏移与移动端交互失效|点位用百分比坐标，添加touch事件适配移动端，增加交互反馈动画|
|多弹窗/面板状态冲突|全局状态管理对象统一控制，封装open/close函数，ESC键绑定关闭所有弹窗|
|长文本与图片加载性能差|图片懒加载+WebP压缩，长文本折叠展示，用requestAnimationFrame优化渲染|
|主题色修改与样式复用难|CSS根变量统一管理颜色，提取.card-base等通用类，采用BEM命名法避冲突|
|IE11兼容问题|CSS属性降级，JS语法转ES5，引入es5-shim垫片，非必要动画功能降级|

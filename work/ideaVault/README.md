# 一、项目概述

本项目是一套基于Web技术构建的福州大学校园相关展示网站，涵盖校园红色打卡点、红色校史、计算机与大数据学院简介及重点实验室等核心模块。网站以"传承红色基因、展示校园风采、普及学术资源"为核心目标，采用原生前端技术栈构建，具备响应式布局、丰富交互效果及良好的用户体验。

# 四、核心模块功能说明

## 4.1 校园打卡点模块（checkin.html）

基于交互式地图的校园点位展示系统，聚焦红色教育点位与特色风景，提供打卡记录与统计功能，打造沉浸式校园探索体验。

### 核心功能

1. **交互式校园地图**：标注6个点位（2个红色教育点位、4个常规特色点位），支持悬浮显示名称、点击查看详情，点位添加脉冲动画增强视觉提示。

2. **打卡点筛选与展示**：采用卡片式布局，支持"全部、红色点位、常规点位、热门点位"四种筛选模式，卡片悬停时呈现3D变换效果。

3. **打卡统计与记录**：顶部统计区域展示总打卡点数、已打卡数、完成率及最受欢迎点位，通过localStorage实现打卡记录本地存储，刷新页面不丢失数据。

4. **详情弹窗**：点击地图点位或卡片打开弹窗，展示点位开放时间、位置、介绍等详细信息，提供"标记已打卡"功能及打卡小贴士。

### 设计特色

采用玻璃拟态设计、动态渐变背景、粒子动画等视觉元素，按钮点击添加水波纹反馈，元素进入视口时触发渐显动画，提升交互体验；响应式布局适配不同屏幕尺寸。

## 4.2 红色校史展示模块（history.html）

以时间轴为核心的校史展示系统，系统呈现福州大学1958年建校以来的红色发展脉络与精神传承，结合地图交互与多媒体展示增强内容表现力。

### 核心功能

1. **红色校史时间轴**：按"1958年建校奠基、1978年改革传承、2008年校史馆落成、2021年数字化建设、未来展望"五大节点分层展示，桌面端左右交替布局，移动端改为单列布局。

2. **红色打卡点展示**：卡片形式呈现校史馆等打卡点信息，包含地址、开放时间及特色说明，支持点击查看详情。

3. **校园地图交互**：标注打卡点地理位置，支持悬停显示简介、点击查看详情，适配移动端触摸事件。

4. **多媒体预览与内容管理**：图片支持点击放大查看，长文本默认折叠展示，支持展开/收起及弹窗放大阅读，ESC键可一键关闭所有弹窗及展开面板。

## 4.3 学院简介模块（college.html）

全面展示福州大学计算机与大数据学院基本信息、师资力量、科研平台等内容，提供清晰的导航与丰富的多媒体展示功能。

### 核心功能

1. **页面导航**：顶部导航栏包含"首页、教师风采、实验室、返回总首页"等导航项，支持锚点平滑跳转。

2. **学院信息展示**：包含学院简介介绍段落、核心统计数据（教师数、学生数、科研项目等），图文结合呈现学院概况。

3. **多媒体展示**：展示三张学院相关图片，支持点击放大预览，实现图片懒加载优化性能。

4. **师资与科研预览**：教师风采展示区呈现多名教授信息卡片，重点实验室预览区展示实验室基本信息，支持跳转至详细页面。

## 4.4 重点实验室模块（lab.html）

聚焦学院27个重点实验室的分类展示与详情查询，支持按研究领域快速筛选，为用户提供实验室研究方向、负责人、科研成果等核心信息。

### 核心功能

1. **实验室分类筛选**：左侧分类栏包含人工智能类（9个）、网络计算类（6个）、数据科学类（5个）、信息安全类（4个）、集成电路类（3个）五大类别，点击分类动态生成对应实验室卡片列表。

2. **实验室列表展示**：默认显示占位提示，筛选后以卡片形式展示实验室基本信息，卡片悬停有阴影效果增强交互感。

3. **详情弹窗功能**：点击卡片打开弹窗，展示实验室名称、分类标签、地点、联系人、联系方式、研究方向介绍及科研成果，支持外部链接跳转。

4. **响应式适配**：弹窗及页面布局适配PC与移动端，分类筛选逻辑通过data-category属性实现自动过滤，提升维护效率。

# 五、技术栈与实现亮点

## 5.1 核心技术栈

- **前端基础**：HTML5（语义化结构）、CSS3（Flexbox/Grid布局、CSS变量、关键帧动画）、原生JavaScript（DOM操作、事件处理、本地存储）。

- **辅助资源**：Font Awesome图标库（提供通用图标）、SVG（背景纹理）、localStorage（本地数据存储）。

- **性能优化**：Intersection Observer（滚动动画、图片懒加载）、WebP图片压缩、requestAnimationFrame（渲染优化）。

## 5.2 实现亮点

1. **视觉设计统一性**：全网站采用福大红色与校徽蓝色为主辅色调，通过CSS根变量统一管理，修改变量即可快速替换主题色；玻璃拟态、3D变换等设计元素贯穿各模块，保持风格一致。

2. **交互体验优化**：所有交互均添加平滑动画反馈（如按钮水波纹、卡片悬停效果、页面滚动渐显）；封装统一弹窗组件，解决多弹窗状态冲突问题，支持ESC键全局关闭。

3. **响应式适配方案**：设768px（移动端/平板）、1200px（桌面端）双断点，通过CSS变量控制间距与布局，时间轴、地图点位等复杂组件适配不同屏幕尺寸。

4. **性能与兼容性保障**：图片懒加载+压缩减少加载耗时，长文本折叠展示优化渲染；兼容Chrome、Firefox、Safari、Edge最新版本，IE11通过属性降级与垫片实现基本兼容。

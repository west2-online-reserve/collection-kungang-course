# 福州大学校史馆网页设计（最终版）

## 项目简介

这是一个展示福州大学校史与红色文化的网页设计项目，包含时间轴、图片展示和简介模块。**所有图片均来自福州大学官方网站**，确保内容的真实性和权威性。

## 项目结构

```
FZU_History_Museum_Final/
├── index.html # 主页面文件（已更新，包含高端UI组件）
├── css/
│ └── style.css # 样式文件（包含玻璃拟态、渐变流光等特效）
├── js/
│ └── script.js # 交互脚本
├── images/ # 图片资源文件夹
│ ├── logo.png # 福州大学校徽
│ ├── museum_banner.png # 校史馆/全景横幅（用于页脚装饰）
│ ├── campus_overview.jpg # 校园全景
│ ├── campus_building1.jpg # 校园建筑1
│ ├── campus_building2.jpg # 校园建筑2
│ ├── campus_landscape.jpg # 校园景观
│ ├── campus_activity1.png # 校园活动1
│ └── campus_activity2.jpg # 校园活动2
└── README.md # 项目说明文档
```

## 功能特点

### 1. 导航栏
- 固定顶部导航，包含福州大学校徽logo
- 支持平滑滚动定位
- 响应式设计，移动端支持汉堡菜单
- 滚动时导航栏样式变化

### 2. 主视觉区域（Hero Section）
- 全屏展示，使用真实校园图片作为背景
- 渐变遮罩突出文字内容
- 滚动指示器

### 3. 校史概览
- 四个关键数据卡片展示
- 悬停动画效果

### 4. 校史时间轴
- 垂直时间轴设计
- 重要历史节点展示（1958、1978、1996、2017、2024）
- **每个节点使用真实校园图片**
- 图片悬停放大效果
- 滚动动画效果

### 5. 红色文化模块
- 三个红色文化主题卡片
- **使用真实校园活动图片**
- 图片悬停放大效果
- 响应式网格布局

### 6. 三种精神
- 展示福大三种精神：艰苦奋斗、严谨求实、勇于拼搏
- 代表人物介绍
- 卡片式设计

### 7. 官方风格页脚（含联系方式）
- **全景横幅**：顶部展示校史馆全景大图
- **官方红底**：采用福大标准深红背景
- **布局设计**：左侧品牌标识，右侧详细联系信息（地址、邮编、官网）

## 设计特色

- **色彩方案**：以红色为主色调（#C8102E），体现红色文化主题，搭配金色（#FFD700）作为强调色
- **真实图片**：所有图片均来自福州大学官网，确保内容真实可靠
- **响应式设计**：支持桌面、平板、手机等多种设备
- **动画效果**：平滑滚动、淡入动画、图片悬停效果等
- **用户体验**：清晰的导航、直观的布局、流畅的交互

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可查看
2. 建议使用现代浏览器（Chrome、Firefox、Edge、Safari等）
3. 所有图片已包含在 `images/` 文件夹中，无需额外下载

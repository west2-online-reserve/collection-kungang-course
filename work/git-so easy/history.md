# 福州大学校史馆
## 校史与红色文化展示

### 时间轴·校史沿革
- **1958年**
  - **图片**：![1958年福州大学建校](establish.jpeg)
  - **大事记**：1958年福州大学建校
  - **简介**：福州大学正式建校，作为福建省属重点大学，肩负起为地方经济建设和社会发展培养人才的使命。建校初期，学校在艰难的条件下起步，汇聚了一批热心教育事业的学者，奠定了办学的基础。
- **1960年代**
  - **大事记**：1960年代福州大学工科实验楼建成
  - **简介**：学校聚焦工科优势学科建设，同时融入红色教育理念，组织师生参与社会实践与爱国教育活动，将红色文化基因融入人才培养过程。
- **1978年**
  - **大事记**：1978年福州大学恢复高考后首届新生入学
  - **简介**：改革开放后，学校迎来发展新机遇，恢复高考后的首届新生入学，学科建设与人才培养规模逐步扩大，红色文化教育也在新时期焕发出新的活力。
- **2005年**
  - **图片**：![2005年福州大学旗山校区启用](qishanxiaomen.jpeg)
  - **大事记** ：2005年福州大学旗山校区启用
  - **简介**：福州大学旗山校区正式启用，校区建设融入了红色文化元素，如校史纪念馆、红色文化长廊等，为校史与红色文化的展示和传承提供了新的载体。
- **2017年**
  - **大事记**：2017年福州大学入选“双一流”建设高校
  - **简介**：福州大学入选国家“双一流”建设高校名单，学校在发展过程中始终坚守红色初心，将校史积淀与红色文化传承相结合，助力学校在新时代实现高质量发展。
- **2023年**
  - **图片**：![2023年福州大学建校65周年校庆活动](65years.png)
  - **大事记**：2023年福州大学建校65周年校庆
  - **简介**：福州大学迎来建校65周年校庆，期间举办了校史展览、红色文化论坛等系列活动，系统梳理校史脉络，弘扬红色文化精神，凝聚全校师生校友的奋进力量。

### 红色文化·福大印记
- **福大红色基因溯源**
  - **图片**：![福州大学红色文化教育基地](rededucatejidi.jpeg)
  - **简介**：福州大学的红色文化基因源于建校初期的爱国办学传统，建校以来，学校始终坚持党的领导，传承红色薪火。老一辈福大人秉持爱国奉献精神，在教学、科研中践行红色使命，为学校留下了宝贵的红色精神财富。
- **师生红色实践**
  - **图片**：![福州大学师生红色研学活动](redacticity.jpeg)
  - **简介**：学校常态化组织师生开展红色研学、志愿服务等实践活动，走进革命老区、红色纪念馆，实地感悟红色文化，将校史中的红色故事与社会实践相结合，让红色文化在青年学子中代代相传。

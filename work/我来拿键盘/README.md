# 校园打卡系统

欢迎来到校园打卡系统项目！该应用程序允许用户探索校园历史，在各个地点打卡，并参与互动游戏。

## 项目结构

项目组织在以下目录中：

- **`Backend`**: 使用 Go 构建的服务器端应用程序。它处理 API 请求、数据库交互 (MySQL) 和缓存 (Redis)。
- **`Frontend`**: 使用 Vue 3 和 Vite 构建的客户端应用程序。它为系统提供用户界面。
- **`Deployment`**: 使用 Docker Compose 和 Cloudflare Tunnels 部署应用程序的配置文件。
- **`Data`**: 数据库初始化脚本。

## 快速开始

要快速启动并运行系统，请参阅 [部署文档](./Deployment/README.md)。

## 功能特性

- **用户认证**: 注册、登录和管理用户资料（头像、密码、昵称）。
- **历史事件**: 查看和管理历史事件及时间轴。
- **校园打卡**: 在特定地点打卡，发表评论并查看弹幕。
- **学院互动**: 评论和点赞与不同学院相关的帖子。
- **游戏**: "合成大西瓜" 游戏及排行榜。
